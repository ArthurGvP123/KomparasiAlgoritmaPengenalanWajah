python 01_merapikan_dataset.py --dataset-name Dosen --size 112 --mode none

Note:
a. mode both untuk mensortir foto dataset menjadi folder gallery dan probe
b. mode gallery untuk mensortir foto dataset ke folder gallery saja
c. mode none membuat program tidak melakukan sortir, melainkan hanya direktori murni dataset saja
d. default ukuran fungsi norm_crop adalah 112 dan 224, jika ingin diluar dari itu maka akan dipilih ukuran terdekat kemudian dilakukan resize
